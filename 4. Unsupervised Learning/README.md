# Unsupervised Learning

This notebook introduces unsupervised learning techniques used to identify patterns and structure in data without predefined labels.

The focus is on grouping similar observations and understanding relationships within datasets.

---

## Overview

Unlike supervised learning, unsupervised learning does not rely on labeled outcomes.

Instead, it is used to explore data, detect patterns, and segment observations based on similarity.

These methods are commonly applied in scenarios where the underlying structure of the data is unknown.

---

## Methods Covered

- K-Means Clustering  
- Hierarchical Clustering  

---

## Key Concepts

- Clustering and similarity  
- Distance metrics  
- Choosing the number of clusters  
- Interpreting cluster results  
- Limitations of unsupervised methods
